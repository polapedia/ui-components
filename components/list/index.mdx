# List

Several related options grouped together vertically.

The List component is a vertical stack of ListItem rows.  
ListItem supports optional `leftIcon`, optional `rightIcon`, and optional `description`.

## Interaction & Behavior

- Use `<button>` semantics for clickable rows (keyboard + accessibility friendly).
- Use `isSelected` to reflect selection state visually.
- Use `disabled` to block interaction.

## Usage Guidelines

- Use List for grouping related options or navigation items.
- Prefer short titles; long text will truncate.
- Use `description` only when the item needs extra context.
- Use icons sparingly to maintain scannability.

---

## Accessibility

- ListItem uses a semantic `<button>` element.
- Focus styles are visible via `focus-visible` ring.
- Avoid putting nested interactive elements inside ListItem.
