# Time Picker

Time Picker allows users to select a specific time from a scrollable dropdown list.  
It features a **two-step selection process** (choose time â†’ confirm with "Select") to prevent accidental changes.

Key features:

- **Custom Format:** Uses `HH.mm` format (e.g., `14.30`).
- **Dynamic Intervals:** Supports custom minute intervals (15m, 30m, 60m, etc).
- **Safe Selection:** Requires explicit confirmation via the "Select" button.

## Usage guidelines

- **Format:** The component strictly uses dot separators (`12.30`) instead of colons (`12:30`) to match the design system.
- **Intervals:** - Use `30` (default) for general appointments.
  - Use `60` for hourly bookings.
  - Use `15` for precise meetings.
- **Behavior:** The dropdown includes a "Select" button. Clicking a time in the list _stages_ the value; clicking "Select" _commits_ the change to the parent component.
- **Label:** Use the `label` prop to provide context (e.g., "Start Time", "End Time").
