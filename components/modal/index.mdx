# Modal

Modal is an overlay dialog used to confirm actions, show important information, or request user input.  
It supports responsive sizing via `size="sm" | "md" | "lg"`, and includes an optional checkbox and footer actions.

## Accessibility Notes

- Uses `role="dialog"` and `aria-modal="true"`.
- Close on **ESC** is supported.
- Ensure `title` is always present for screen readers.
